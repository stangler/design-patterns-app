# Template Method

## 概要
Template Method（テンプレートメソッド）パターンは、アルゴリズムの骨組みを親クラスで定義し、詳細な実装をサブクラスに委譲するパターンです。アルゴリズムの構造を変えることなく、特定のステップをサブクラスで再定義できます。コードの重複を避け、一貫性のある処理フローを保証します。

## 主な登場人物
- **AbstractClass（抽象クラス）**: テンプレートメソッドと、サブクラスで実装される抽象メソッドを定義する
- **ConcreteClass（具体的クラス）**: 抽象メソッドを実装し、アルゴリズムの特定のステップをカスタマイズする

## 使う場面
- データ処理パイプライン（読み込み→変換→出力）
- フレームワークのライフサイクルメソッド（初期化→実行→終了処理）
- レポート生成（ヘッダー→本文→フッター）
- テストケースのセットアップ・実行・クリーンアップ
